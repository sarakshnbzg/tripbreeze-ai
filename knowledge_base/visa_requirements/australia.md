## Australia
- **US citizens:** ETA (Electronic Travel Authority) required. Apply via app. Cost: AUD 20. Valid for 12 months, stays up to 3 months.
- **Canadian citizens:** ETA required. Same conditions.
- **EU citizens:** ETA or eVisitor visa (free for most EU countries). Valid up to 3 months.
- **UK citizens:** eVisitor visa (free, valid up to 3 months).
- **Japanese citizens:** ETA required.
- **Indian citizens:** Visitor visa required. Apply online. Processing: 20–30 days. Cost: AUD 190.
- **Chinese citizens:** Visitor visa required. Cost: AUD 190.
- **South African citizens:** Visitor visa required.
- **Documents needed:** Passport valid 6+ months, proof of funds, return ticket, health insurance recommended.
