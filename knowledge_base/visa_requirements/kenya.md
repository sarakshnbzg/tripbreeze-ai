## Kenya
- **US citizens:** eTA (Electronic Travel Authorization) required. Apply online. Cost: $30. Valid for 90 days.
- **Canadian citizens:** eTA required. Cost: $30.
- **EU citizens:** eTA required. Cost: $30.
- **UK citizens:** eTA required. Cost: $30.
- **Australian citizens:** eTA required. Cost: $30.
- **Indian citizens:** eTA required. Cost: $30.
- **Chinese citizens:** eTA required. Cost: $30.
- **Documents needed:** Passport valid 6+ months, return ticket, proof of accommodation, yellow fever certificate required.
- **Special note:** Kenya moved to eTA system in 2024, replacing visa-on-arrival.
